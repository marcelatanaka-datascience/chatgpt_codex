# Module Management

Estrutura inicial do projeto React para o módulo de Module Management, baseada no Storybook da Coca-Cola Femsa.

## Desenvolvimento

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Build de produção:
   ```bash
   npm run build
   ```
